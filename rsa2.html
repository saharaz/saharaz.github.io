This is what Korn and I made. It allows the user to encrypt a message (if they have a public key) and decrypt a message (if they have a private key). We started with the GUI of the project, which was the easiest part. Afterwards, we had to make the buttons actually do stuff. We used some of the code we already made in a previous in class assignment, with a few adjustments. Some difficulty we had was our entry boxes. At one point, we had our e and n entry boxes mixed up, so messages wouldn't encrypt correctly. We also had trouble decrypting another group's message. We still don't know why. Other than that, the project went pretty smoothly for the most part.
